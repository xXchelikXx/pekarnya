# Сеть "Тьерри"
Это сайт сделан по макету франшизы "Тьерри".

На сайте можно увидеть продукцию пекарни, цены на неё и скидки.

## Как запустить
Для запуска сайта вам понадобится Python третьей версии.

Скачайте код с GitHub. Запустите его:
```
python main.py runserver
```
Затем перейдите по [этой ссылке](http://127.0.0.1:8000/)
